# 通用论文大纲

# 关于论文
- 广读论文，总结出关键问题，就已经写完一半。
- 针对关键问题，提出 合理有效的解决方法，就相当于完稿。
- 先定初稿，走通整个流程。（记住：修改 总是比 新写 容易）


# 说明
- 最近在写大论文，抽空mark一下 写小论文的思路和步骤。
- 一种论文风格，无分好坏，仅限参考。
- 待更新 示例论文批注结构。
